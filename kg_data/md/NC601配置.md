OH指示灯：当Modem拨号时，OH灯亮，挂机后OH灯灭。

SD指示灯：当电脑终端传送数据给Modem时，SD灯闪烁。

RD指示灯：当Modem向其连接的设备传送数据时，RD灯闪烁。

TS指示灯：当TS灯闪亮一次，Modem进入工作状态。

# 一. IP配置（远程拨号）

在 NC601 通讯服务器产品说明书上查看 NC601 通讯服务器的预设的 IP 地址（通常默认为 192.168.0.233），在 Windows 上添加对应的网段，通过 telnet 进入段 NC601 通讯服务器

# 1、设置NC601通讯服务器

IP 地址[100 100 100 20]，以太网子网掩码[255.255.255.0]

![](images/625129288c56e8844446905aa642a68cbeaba9f795f08a1a666fc1ea85a3330d.jpg)

2. 按“ESC”返回主菜单界面，选择“串口”修改：端口[1]波特率[115200]数据[8]停止[1]校验[n]流量控制[none]类型[RS232]

![](images/0e4d92c87902d06f9f280e60cdf4a7ad867177d8a31916a1457e16da340d9561.jpg)

3.进入“串口- - - - - - - - - - 高级”进行配置，按下图所示设置各项参数值

![](images/d7a6774418663d9c6ef2a875eaa8529db3f10b2e86173e108fc35d872543a5a7.jpg)

4. 选择“模式”，将“工作模式”设置为“PPP daemon”。

![](images/de2d3602eface3b49f1aa213a511030d7d8aa78f3b43d3d06b8dd153857bc308.jpg)

5. 选择：[详细参数设置...], 参照下图所示修改各项参数值，其中“本地 IP 地址”指的是“通地址”指的是 NC601 通讯服务器的 IP 地址；“对端 IP 地址”是当远程终端通过 MODEM 拨号与

NC601 通讯服务器连接时使用的 IP 地址，必须时使用的地址，必须和 NC601 通讯服务器的 IP

地址为同一网段；“MODEM初始命令为” MODEM初始命令为为ats0=1，设置modem为自动应答。

![](images/c54d3f52c416c19e2f8bab17702f3327feda5a6ef54046c14207ba76c334a039.jpg)

6. 选择“用户”，添加”用户名“和“密码”都为aith的用户。

![](images/35e608509eb9a23786384494db88a115d8a1e7e777be0ec765cf0e24f3495f8d.jpg)

7. 选择综合-->保存当前设置，然后选择“重启设备”。

![](images/2eca55f8c8b21d6bbd42b7962e14ec1a4a9de0a5c107d2ca34412c3283ffbec4.jpg)

# 二. 连接 GPS 时的配置:

1. 设置 NC601 通讯服务器 IP 地址[100.100.100.18]，以太网子网掩码[255.255.255.0]；

![](images/9887c87fabbc592ec4b075e747a0f8c7532313035223fb4042633421604c023d.jpg)

2. 选择串口修改：端口[1]波特率[115200]数据[8]停止[1]校验[n]流量控制[none]类型[RS232]

![](images/37e27f4b1577e8d1c2010259768c385bfd2c67926006434c1f96775383e66d0f.jpg)

3. 进入“串口 - - - - - - - 高级”进行配置，按下图所示设置各项参数值：

![](images/69d5162156e8b3877ade6b77b1d38645758988437089e0e6da3ba6c7c46967a1.jpg)

4. 选择“模式”，将“工作模式”设置为“TCP/UDP Socket”;

![](images/e00a4fcdff8a67585b857cb30cf680534315465ebad7812f71ef17158115d425.jpg)

5. 选择[详细参数设置]参·5、选择：[详细参数设置...],参照下图所示修改各项参数值：在TCP数据项，选择telnet设置；在本地端口项输入10001；在协议项，选择UDP协议，在对端主机

项和端口项分别输入使用 UDP 通讯时，与 NC601 通讯的设备的 IP 地址和端口如果需要将串口设地址和端口。如果需要将串口设备发出的数据转发给两个接收 UDP 数据的设备，需要在会话数项，选择 2，屏幕下方会出现第二个设备的地址、端口的设置项。其它设置项均选缺省值。

例如：主机配置为 100.100.100.1  
100 100 100 2 ; 100.100.100.2  
- 会话数项：选择2

![](images/0e5cfc89ed4aabca87a3bc9841e6cc8ac05c71b8b25f1d40770727e54b3f9406.jpg)

6. 选择“综合”修改：保存设置。

![](images/94cc9d5116c53dd78412e7a653871c3646d761a567b0b45dc3c1697468ad3b1c.jpg)

7.再选择“重启设备”。

![](images/ee2ff8402743cc05b4e5ac9bc0a5e635c0b198e96efccb212376a3be8cb786de.jpg)
